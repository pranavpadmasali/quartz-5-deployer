If you can read this, **your blog is working**. 🎉

This home page is just a Markdown file: [`content/index.md`](https://github.com). Open it in your repo, edit the text, commit, and your change goes live in a minute or two.

## Start here

1. **Name your blog** — open `quartz.config.yaml` and change `pageTitle` (that's the biggest thing).
2. **Write a post** — add a `.md` file inside `content/posts/`. Copy an existing one to see the format.
3. **Edit this page** — replace everything here with your own introduction.

> [!tip] Everything is Markdown
> Folders become sections, files become pages, and `[[wikilinks]]` connect them. No HTML or build commands required — just write.
